Inorder to run the script and generate recommendations, follow these steps.

If you already have R and R Studio installed on your machine.

* Load the script into the R Studio and compile it. 
* Use the function with a userID to generate recommendations using different techniques.

If you dont have R and R Studio installed on your machine.

* Download and install R from http://cran.r-project.org/mirrors.html.
* Download and install RStudio from http://www.rstudio.com/products/rstudio/download/
* Load the script into the R Studio and compile it. 
* Use the function with a userID to generate recommendations using different techniques.
